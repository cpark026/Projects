  • Use Load Sample Data to load demonstration data

About

This map visualizes predicted crash hot spots across Virginia using machine learning models trained on historical crash data from 2017-2025. Predictions are available for each hour of the day.

Data Source: Virginia Crash Records 2017-2025 | Model: R Studio ML Pipeline
